## サイト概要
### サイトテーマ
・クリエイターやアーティスト同士が繋がれるSNS

### テーマを選んだ理由
・自分が過去に何度も音楽や動画編集などに興味を持ってきましたが、その度に身近に相談できる人がいなく寂しい思いをしたため、
　クリエイティブ系に特化したSNSがあると便利だなと思ったためです。

### ターゲットユーザ
・クリエイティブ系の作業をしている、あるいは始めたい人全般
　（プロ、アマチュア問わず）

### 主な利用シーン
・同じ作業をする仲間を見つけたい時
